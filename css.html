CSS - понятия и основы:

CSS - это каскадные таблицы стилей, которые позволяют разнообразить контент создаваемый одним html. С помощью данной технологии можно менять цвета шрифтов, фонов; еще можно задавать позицию элементов на сайте.

Сейчас, сайт без CSS не мыслим, ведь с появлением новых требований и стандартов, сайт без CSS - не валиден.

Рассмативая технологию каскадных таблиц стией, можно сделать вывод, что существуют несколько видов таких таблиц.

Виды таблиц стилей:

Внешняя таблица стиля - это файл с расширением .css который подключается в теге <link>, внутри тега <head>, синтаксис: <link rel="stylesheet" href="css/style.css">.

Внутренняя таблица стилей - также встраиваются в тег <head>, но структура кода у них другая: <style> </style>. То есть внутри данного тега уже прописываются свойства для элементов. Но в отличии от внешней таблицы, она работает лишь на той страницы, где указана.

Встроенные стили - это атрибут для элементов HTML - кода, который позволяет задать стиль, не прибегая к использованию css файла. Например, <p style="max-width: 500px;"> </p>

Что такое селектор:

Селекторы - это структурные элементы web - страницы, которые позволяют создать CSS правила для всей страницы или элемента в частности. К селекторам относятся class, id, псевдоклассы и псевдоэлементы.

Виды селекторов:

Общий селектор - это селектор позволяющий задать стили к всем элементам на странице. Но с данным селектором нужно быть аккуратным. Пример использования общего селектора: * { padding: 0; margin: 0; }, данный стиль убирает стандартные отступы для элементов.

Селектор для отдельного элемента - это селектор, который позволяет стилизовать элемент определенного типа, будто это элемент текта p или целый блок div. Тоесть если задать стиль определенному элементу, то этот стиль распространится на все элементы этого типа на сайте. Пример использования данного селектора: p { font-size: 20px; }.

Селектор class - это селектор, который позволяет стилизовать элемент, по заданному ему имени в атрибуте class. При записи стилей, перед классом элемента ставится точка. Например, если тегу p задать class = "subtitle", то его можно стелизовать по этому классу: .subtitle { font-family: sans-serif; }

Селектор id - это селектор, позволяющий стилизовать ровно один элемент на страницы. Тоесть селектор id с определенным именем нельзя задавать элементу, если на страницы есть элемент с таким же id, также имя данного селектора должно быть уникальным. При записи стилей, перед элементом ставится #. Например, #header { position: absolute; top: 0;}
