Элементы и теги элементов - HTML:

HTML - элемент - это основа всего HTML кода. Каждый HTML документ состоит из так называемого дерева элементов, которые являются определителями, какой контент должен быть в том или ином месте на этом дереве.

Тег - это то, из чего состоит любой элемент в структуре HTML дерева. Тег является необходимым звеном в построении структуры элемента, он открывает элемент, а затем закрывает его. Синтаксис тегов прост, они могут быть пустыми, обычными, с неформатированным текстом, выводящие неформатированный текст.

Синтаксис тегов:

Синтаксис пустых тегов - <...>

Синтаксис обычных тегов - <...> </...>

Синтаксис тегов с неформатированным текстом - <script> </script>

Синтаксис тегов выводящх неформатированный текст - <textarea>

<area> Делает активными, некоторые области изображения.
