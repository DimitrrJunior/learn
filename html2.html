Что такое атрибуты?

HTML - атрибут - специальное слово или выражение, которое позволяет контролировать состояние того или иного элемента, меняет его функционал или создает новый.

Атрибут имеет имя, исходя из его значения. При добавлении атрибута, стоит учитывать его значение, а также правила синтаксиса. Атрибут обязательно должен содержать служебные символы, такие как "", '', >, /. Атрибуты можно создавать, не учитывая регистр, но всегда используется нижний.

Синтаксис атрибутов:

Синтаксис атрибутов без параметров - пишется просто имя атрибута, например: required

Синтаксис атрибутов без ковычек, например: controls

Синтаксис атрибутов с параметрами в одинарных ковычках, например: type = 'параметры'

Синтаксис атрибутов с параметрами в двойных ковычках, например: class = "название класса"

Таблица глобальных атрибутов:

Помимо атрибутов, которые можно присвоить к определенному элементу, существуют атрибуты, которые можно присвоить ко всем элементам.

Атрибут

Описание атрибута
class Классы необходимы для назначения имени элемента. Они предоставляют другим технологиям, таким как: css, JavaScript, доступ к данному элементу, и возможность модифицировать его. Классам можно задать несколько имен, что упрощает создание сайта. Синтаксис: <div class = "имя"> </div>.
id Служат для индентификации элемента. Также как class используются для модификации или стилизования, но отличаются тем, что они являются некими константами, им можно задать одно имя. Синтаксис: <div id = "имя"> </div>.
hidden Скрывает элемент на сайте, но по прежнему является блоком. Синтаксис: <div hidden> </div>
accesskey С помощью данного атрибута можно создать сочитание клавиш клавиатуры, которое позволит сфокусироваться на элементе, к которому этот атрибут присвоен. Синтаксис: <div accesskey = "сочитание клавиш"> </div>.
autofocus Позволяет сфокусироваться на элементе, к которому присвоен данный атрибут. В отличии от accesskey , он делает это в автоматическом режиме и при загрузке страницы. Синтаксис: autofocus.
contenteditable Указывает, доступен ли элемент к редактированию пользователю. Принимает значения True или False. Синтаксис: <div contenteditable = "true or false"> </div>
lang Используется для указания основного языка содержания элемента и его атрибутов. Синтаксис: <div lang = "en or ru"> </div>
title Используется как подсказка. При наводе мышкой на элемент выплывает окно с информацией, написанной в параметрах атрибута. Синтаксис: <div title = "текст"> </div>
translate Указывает hello, следует ли перевести контент содержащийся в элементе. Принимает значения yes или no .Синтаксис: <div translate = "yes"> </div>
slot Назначает слот элементу. Синтаксис: <div slot = "title"> </div>
spellcheck Проверяет элемент на наличие орфографических ошибок. Принимает значения true или false Синтаксис: <div spellcheck = "true or false"> </div>
