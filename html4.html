Тег ссылки и его атрибуты - HTML:

HTML - тег ссылки - элемент на дереве сайта, который представляет собой связь между документом, тоесть страницей сайта, и прочими ресурсами или документами.

Элементы ссылок делятся на разные типы, они могут быть внешними, которые подключают прочие файлы для работы с сайтом, и гиперссылки, которые связывают элемент на данном сайте с другим сайтом или ресурсом.

Синтаксис элементов ссылок:

Синтаксис внешних ссылок - <link>

Синтаксис гиперссылок - <a> </a>

Таблица атрибутов элементов ссылки:

Атрибут

Описание атрибута
href

Вмещает в себя URL - ссылку на другой документ. Пользователь, нажав на ссылку, автоматически переходит в новый документ или ресурс.

Синтаксис атрибута: <a href = "путь до файла, или ссылка на ресурс"> </a>
download

При нажатии на ссылку, файл, который был добавлен внутрь ссылки, будет скачен на комьютер. Это может применятся к фотографиям, картинкам, некторым медиа и видео файлам.

Синтаксис атрибута: <a href = "..." download> </a>
hreflang

Определяет, какой язык присущ связанному документу или ресурсу, поэтому не используется без атрибута href.

Синтаксис атрибута: <a href = "..." hreflang = "en or ru"> </a>
ping

Данный атрибут позволяет отслеживать, перешел ли пользователь по ссылке.
target

Данный атрибут указывает на то в какой вкладке будет открываться ссылка. По стандарту, ссылка открывается в той вкладке, в которой находится пользователь.

Параметры, которые принимает атрибут:

_blank - открывает файл или ресурс в новой вкладке.

_self - ссылка открывается во вкладке, где находится пользователь.

_top - ссылка открывается во вкладке на всю страницу.

_parent - ссылка открывается в родительском элементе.

Cинстаксис атрибута: - <a href = "..." target = "_blank"> </a>
type Определяет тип файла, его расширение.
