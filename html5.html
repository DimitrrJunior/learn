Тег изображения и его атрибуты - HTML:

Тег изображения - элемент на дереве сайта, который представляет собой контейнер для изображений. Данный тег является строчным, тоесть ограничивается пространсвтом внутри блока, которому уже заданы размеры. Поэтому, зачастую такие тэги используют в других тегах, на подобии: div, которые являются блочными.

Данный элемент имеет структуру: <img>. Это пустой тег, в который также можно прописать параметры, или же наложить стили. Также, этот элемент имеет два атрибута при создании src, alt.

Синтаксис элемента изображения:

Синтаксис - <img src = "" alt = "">

Таблица атрибутов элементов изображения:

Атрибут

Описание атрибута
src

Вмещает в себя URL - ссылку до изображения, или ссылку на изображение из стороннего ресурса.

Синтаксис атрибута: <img src = "...">
alt

Размещает в себе контент, который будет виден на сайте в случае, если изображение не подгрузится.

Синтаксис атрибута: <img alt = "...">
height, width

Задают параметры ширины и длинны изображения. Данные параметры измеряются в px - пикселях.

Синтаксис атрибута: <img width = "..." height = "...">
longdesc

Данный атрибут замещает атрибут alt в случае, если в описании к изображению, нужна информация из другого файла, из другого ресурса.

Синтаксис атрибута: <img longdesc = ".../hello.txt">
srcset

Данный атрибут создает список из ресурсов с изображениями, которые будут появлятся в зависимости от размера экрана. Данный тег упрощает адаптивную верстку.

Синтаксис атрибута: <img srcset = "...">
sizes Используется с атрибутом srcset, для указания размеров изображений.

Синтаксис атрибута: <img srcset = "..." sizes = "...">
