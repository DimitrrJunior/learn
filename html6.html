HTML списки:

HTML списки - используются для группировки контента на сайте. Объединяет одинаковые по смыслу элементы. Зачастую, списки используются в создании меню для сайта.

Списки делятся на несколько типов: маркированный (с точками), нумерованный (с цифрами), список для определений. Все типы списков включают в себя тег li, который отвечает за элементы внутри списка.

Синтаксис элемента изображения:

Синтаксис маркированного списка - <ul> </ul>.

Синтаксис нумерованного списка - <ol> </ol>.

Синтаксис списка с определениями - <dl> </dl>.

Таблица атрибутов списка:

Атрибут

Описание атрибута
start

Данный атрибут назначает цифру, с которой будет начинаться отсчет нуменации в нумерованном списке.

Синтаксис атрибута: <ol start = "43"> </ol>
reversed

Говорит браузеру о том, что нужно вести отсчет списка в обратном порядке.

Синтаксис атрибута: <ol reversed> </ol>
type

Данный атрибут задает тип маркера, которым будет нумероваться список.

Синтаксис атрибута: <ol type = " "> </ol>

Значения, которые принимает атрибут:

1 - по умолчанию.

A - создает список с нумерацией из букв в верхнем регистре.

a - создает список с нумерацией из букв в нижнем регистре.

I - создает список с нумерацией из римских цифр в верхнем регистре.

i - создает список с нумерацией из римских цифр в нижнем регистре.
