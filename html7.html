HTML таблицы:

HTML таблицы - используются для группировки информации в строки и столбцы в ячейках. Как обычно таблицы строятся с помощью пересечений столбцов и строк.

таблицы являются контейнером для любых элементов: текста, ссылки, изображения, списки. Восновном таблицы не используются в современных сайтах, так как это усложняет процесс адаптации сайта на других видах устройств.

Синтаксис таблиц:

Синтаксис таблиц сложнее, чем у других элементов, так как он включает в себя еще несколько элементов, таких как:

<table> </table> - объявляет на сайте таблицу.

<thead> </thead> - объявляет шапку таблицы.

<tbody> </tbody> - объявляет тело таблицы.

<tfoot> </tfoot> - объявляет подвал таблицы, в котором подводятся какие - либо итоги.

<th> </th> - объявляет заголовки таблицы, или колонок таблицы, рекомендуется использовать в <thead> </thead>.

<td> </td> - объявляет основной контент ячеек в таблице, рекомендуется использовать в <tbody> </tbody>.

Таблица атрибутов элементов изображения:

Атрибут

Описание атрибута
align

Данный атрибут служит для выравнивания таблицы на сайте.

Синтаксис атрибута: <table align = "right"> </table>
background

Данный атрибут задает изображение заднего фона таблицы.

Синтаксис атрибута: <table background = "путь к изображению"> </table>
bgcolor

Данный атрибут задает цвет заднего фона таблицы.

Синтаксис атрибута: <table bgcolor = "цвет | HEX код"> </table>
border

Данный атрибут задает ширину рамки вокруг таблицы, и между ее элементами.

Синтаксис атрибута: <table border = "значение в px"> </table>
bordercolor

Данный атрибут задает цвет рамки таблицы.

Синтаксис атрибута: <table bordercolor = "цвет | HEX код"> </table>
сols

Данный атрибут задает число колонок в таблице.

Синтаксис атрибута: <table cols = "число"> </table>
height

Данный атрибут задает фиксированную высоту таблицы.

Синтаксис атрибута: <table height = "значение в px"> </table>
width

Данный атрибут задает фиксированную длинну таблицы.

Синтаксис атрибута: <table width = "значение в px"> </table>
summary

Данный атрибут позволяет создать краткое описание для таблицы.

Синтаксис атрибута: <table summary = "описание"> </table>
cellpadding

Данный атрибут позволяет создать отступ от рамки до контента в ячейки таблицы.

Синтаксис атрибута: <table cellpadding = "значение в px"> </table>
cellspacing

Данный атрибут позволяет создать отступ между ячейками таблицы.

Синтаксис атрибута: <table cellspacing = "значение в px"> </table>
