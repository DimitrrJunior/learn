Описание web - технологий

HTML

Язык гипертекстовой разметки, служит для просмотра web - страниц браузера. Является скелетом любого браузера, его элементы являются строительными блоками любого современного сайта. Последняя стандартная версия технологии - HTML 5.

Без дополнительных технологий сайт выглядет не красиво и совершенно не современно. Поэтому на помощь приходят такие технологии как CSS, Java Script...

Перейти к документации HTML

CSS

Язык описания web - документа, написанного с помощью языка гипертекстовой разметки HTML. CSS используется для задания цветов, шрифтов, стилей и других аспектов внешнего вида web - страниц.

CSS широко используется для того, чтобы разделить понятие структуры страницы, которую представляет - HTML, от описание внешнего вида, которым занимается сам CSS. Если мы бы хоть на секунду представили, что данной технологии не существует, то мы бы не увидели современные сайты в привычном нам виде.

Перейти к документации CSS

JavaScript

Является мультипарадигменным языком программирования. Поддерживает объектно-ориентированый, императивный, функциональный стили.

Широкое применение javaScript нашел в web - разработке, как объектно-ориентированый язык программирования, так - как работа с данным языком, лежит на основе объектов или классов.
